# 문제점
1. 농산물 품질 인증 제도 7가지 + HACCP 등 식품 인증 : 소비자 선택 어려움
2. GAP(우수농산물 판매인증) - 친환경과 차별성 문제
3. 유통
 - 친환경 농산물이 소비되지 않으면 관행 농산물로 팔리고 있는 현실
 -> 학교과 군대 급식으로 배급이라도 해달라 : 정책적 지원 요구
4. 친환경 농업의 생태적 중요성을 소비자가 알 수 있도록
5. 결과가가 아닌 생태를 지키는 과정 중심의 인증으로 변경 요망
6. 반복되는 부실인증 사건


# 뉴스
1. 살충제 계란
 - 친환경 농축산물 안정성에 대한 국민들 우려 커짐
 -> 민간 인증기관들의 단속 강화
 -> 친환경 인증 취소 건 늘어남

2. 취소 농가 중 자발적 의도가 아닌, 타의적 희생양
 - 인근 논에서 농약이 날아오거나, 항공방제로 인한 비의도적 검출
 -> 그러나 항공방제로 인한 경우더라도, 이와 관련된 서류를 납입하더라도
    인증이 취소되고 학교 급식에 출하가 정지됨.
 -> 불만과 억울함 호소
3. 구조적 부패 비리 : 부당하게 취소되는 경우가 없도록
 - 안정성 검사 엉터리
 -> 검찰 수사  : 합법적인 검사 기관의 결정 불복하면 업무방해로 혐의 입건 가능성
 -> 수사요청 각하
 -> 언론에 들어감
 -> 국립 농산물 품질 관리원에서 수습차원에서 무혐의 결정
 -> 개선 : 비의도적 인증 위반 곧바로 인증 취소하던 것을 1차 시정명령
	  이후 2차 인증 취소하는 방식으로 개정 작업 진행중.
